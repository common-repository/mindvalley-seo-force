Force authors to fullfill SEO requirements before publishing posts.


== Description ==

This shortcode will optionally force SEO metadata to be entered before allowing a post to be published.

Works wonders if you have a dedicated SEO team that works with another team of authors.

It will :-

1. Notify a list of emails whenever there's a new post creation.
1. Display a to-do list to check off before publishing.
1. Autocompletes the Title and Keywords field with predefined keywords. 

Plugin Supports :-

* All In One SEO Pack


== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory or Download it using the 'Plugins' > 'Add New' menu
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy!
